## 📝 I. Overview

Dự án tập trung vào việc **khai thác và phân tích dữ liệu từ bộ dữ liệu có cấu trúc** nhằm hiểu rõ đặc điểm dữ liệu và trích xuất insight phục vụ phân tích.

---

## ❗II. Problem

Dữ liệu có sẵn nhưng:
- Chưa được phân tích và khai thác hiệu quả
- Khó hiểu cấu trúc và mối quan hệ giữa các thuộc tính
- Thiếu truy vấn để hỗ trợ trích xuất thông tin
👉 Cần thực hiện truy vấn và phân tích để phục vụ **data exploration**

---

## 🛠️ III. Approach / Methodology

**1. Data Understanding**
- Mô tả các thuộc tính của bộ dữ liệu
- Xác định kiểu dữ liệu và mối quan hệ giữa các trường

**2. Data Querying**
- Viết các câu truy vấn SQL để trích xuất dữ liệu
-Lọc, nhóm và tổng hợp dữ liệu theo yêu cầu phân tích

**3. Data Analysis**
- Phân tích kết quả truy vấn để xác định pattern và insight
- So sánh dữ liệu theo các tiêu chí khác nhau

---

## 🚀 IV. Results & Impact

- Hiểu rõ cấu trúc và đặc điểm của bộ dữ liệu
- Trích xuất được các insight phục vụ phân tích
- Hỗ trợ quá trình ra quyết định dựa trên dữ liệu
